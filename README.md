⚡ AI Powered College Fest Management Portal

A futuristic, responsive, real-time college fest management prototype built for hackathon demonstration and smart campus event workflows.

🌌 Overview

FestFlow AI is a high-fidelity hackathon prototype designed to modernize how college festivals and technical events are managed.

The platform provides:

🎟 Smart participant registration
🧠 AI-powered event rule generation
📡 Real-time announcements
🏆 Dynamic leaderboard system
🔐 Secure admin authentication
📱 Fully responsive UI/UX
🎫 QR-based digital passes
⚡ Futuristic cyberpunk interface
